# Flexlights Firmware

Official firmware releases for Flexlights controller.

This repository contains only compiled firmware files (`.bin`).
Source code is not published here.

## Download latest firmware

Latest firmware release:
https://github.com/kykywkeee/flexlights-firmware/releases/latest

## How to download

1. Open the latest release.
2. Download the `.bin` file from Assets.
3. Do not download "Source code (zip)" or "Source code (tar.gz)" — these are automatic GitHub archives, not firmware files.

## Release contents

Each release includes:

- firmware `.bin` file
- version number
- changelog
- important notes, if needed
