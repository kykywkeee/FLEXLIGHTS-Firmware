FLEXLIGHTS Firmware v1.1

What's new?
1. Motion sensor sensitivity has been changed. The device now reacts to movement more smoothly. 2. Back up your images and playlists. Use Backup to save your files before sharing and syncing with your friends’ devices. 3. Each device now have a group number. Syncing, image uploads, and other commands will only work inside the selected group. 4. Press the light bulb icon next to a device to highlight it. 5. Press the X icon next to a device to turn it off. 6. You can turn off only one device using the button, instead of turning off the whole group. Press 3 times and hold. 7. Your device now keeps the connection automatically. If you see “Connection lost”, first check that the device is turned on, then check the Wi-Fi connection. If everything is okay, the page should refresh automatically. If it does not, refresh the page manually. 8. Some phones can access the internet while connected to the device’s Wi-Fi network. 9. Optimization has been improved, and the page now loads faster. 10. Found bugs and errors have been fixed.
